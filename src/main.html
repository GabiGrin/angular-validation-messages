Angular-validation-messages Demo Page

Basic usage

First 2 fields are required, try submitting the form before they are filled, so see the validation messages in action. The messages are also triggered when a field is blurred but still invalid.

Triggers

You can override the show/hide triggers for the validation messages, to either show them immediately when the user starts the interaction, or only when trying to submit.

Custom message template

You can use a custom message template if wanted, and like all options, you can either set them globally, per form, or per field. This example will apply it to the whole form. In this example we'll also pass a custom class name for our validation messages. And yep, it's not pretty ;)
